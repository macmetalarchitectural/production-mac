Configuration des abonnés

Ce module permet de configurer l'ajout automatique d'abonnés dans les différents documents Odoo:

  • Ventes: soumission, bon de vente (commande) et facture client
  • Achats: demande de prix, commande et facture fournisseurs
  • Contacts
  • Écritures comptable
  • Transferts.

Module(s) requis

Au minimum un des modules touchés

Ventes Achats Contacts Comptabilité Inventaire

Dans le module Configuration, PARAMÈTRES GÉNÉRAUX, repérer la section Messages. Il est possible, pour chacun des documents énumérés, de déterminer quels seront les abonnés créés automatiquement.
Lorsqu'une case est cochée, la personne visée par la configuration (créateur, vendeur, client, fournisseur, etc.) sera ajoutée comme abonnés dans le document.
Au contraire, si la case n'est pas cochée, la personne visée ne sera pas ajoutée comme abonné.

Fonctionnement du module

Dans l'exemple ci-haut, lorsqu'une soumission est créée, seul le créateur du document est abonné alors que dans la demande de prix d'achat, ce sont le créateur et l'acheteur identifié dans le document qui sont tous deux abonnés.
Il faut savoir que le module e3k_followers_mgmt n'affecte pas le comportement natif d'Odoo qui ajoute comme abonné le destinataire d'un message envoyé via Enregistrer une note dans le chatter.
