Stock Block Auto Purchase Order

Résumé

Le module permet de créer un purchace order pour chaque ligne du sale order sauf si l’option « Bloquer ajout produit auto » l’article est cochée, dans le cas contraire le comportement natif odoo sera exécuté.

Module(s) requis

purchase_stock

Étapes de mise en place / configuration du module

  1. Installer le module

  2. Configurer les articles qui sont censés être séparé dans les purchace order

Fonctionnement du module

1. Pour chaque article on a deux options à définir

- Bloquer ajout produit auto

- Unique Purchase Order

Si l’option « Bloquer ajout produit auto » est cochée, l’article sera mis dans un purchace order a part (même si ce produit existe déjà dans un autre PO)

Si l’option « Unique Purchase Order » est cochée, et si l’article sera vendu plusieurs fois dans le même SO, les quantités de l’article seront regroupés dans un seul PO


2. Si l’option « Bloquer ajout produit auto » est cochée, et l’article est ajouté dans purchace order l’information sera affichée dans le PO
